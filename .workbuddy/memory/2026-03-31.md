## 黑屏修复（10:49）

### 问题
打包后启动界面全黑，原因：`getResourcePath` 函数在打包模式下用 `path.join(process.resourcesPath, 'app.asar', relativePath)` 构建路径，`loadFile()` 无法正确解析此格式。

### 修复
`main/src/index.ts`：改为通过 `__dirname` 回溯路径
```js
// 旧写法（有问题）
return path.join(process.resourcesPath, 'app.asar', relativePath);
// 新写法（正确）
return path.join(__dirname, '../../', relativePath);
// __dirname 打包后 = {resourcesPath}/app.asar/main/dist，../../ 退到 asar 根
```

### 结果
重新编译 + 重打包，应用正常启动，不再黑屏。
